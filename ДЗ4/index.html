Универсальный селектор "*"

-может подразумеваться любой элемент разметки HTML-страницы. Выглядит примерно так:

"* {"
"font-size: 16px;"
"}"

Это не будем рассматривать

Cелектор по названию тега

-какой тег так и пишем) Выглядит вот так:

"h1 {"
"color: red;"
"}"

Пример селектора по тегу

Cелектор по классу

-«class» — это атрибут HTML-тега, он может применяться к однотипным элементам. Выглядит вот так:

".main-header {"
"background-color: #99ffcc;"
"}"

Пример селектора по классу.

Cелектор по нескольким классам

-«class» — это атрибут HTML-тега, он может применяться к однотипным элементам. Выглядит вот так:

".main-header.main-experience {"
"font-weight: bold;"
"}"

Пример в синем цвете)

Cелектор по id

-элемент с id может быть в единственном экземпляре на странице согласно правилам семантики. Выглядит вот так:

".#heading {"
"font-size: 35px;"
"}"

Пример селектора по id

Cелектор по вложенности

-чтобы стили для этих тегов использовались корректно, помогут вложенные селекторы. Выглядит вот так:

"post .title {"
"font-size: 10px;"
"}"
